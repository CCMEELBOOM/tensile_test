# tensile_test
Data processing for tensile testing of 6061 T6 Aluminum and 1018 Cold Worked Steel <br>

This repository contains data collected for tensile tests performed on four samples of 6061 T6 Aluminum and 1018 Cold Worked Steel ( 2 and 2 accordingly). <br>

To be able to run the python scripts attached make sure to download the .csv files within the same local repository.
